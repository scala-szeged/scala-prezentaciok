# Scala előadások

JDK 8 és Scala installálása kell az előadások lejátszásához

Scala -t innen lehet letölteni: https://www.scala-lang.org/download/

A következő utasítások szükségesek a prezentáció elindításához:

cd scala-alapok  

Windows: <br>
Scala-alapok.bat  

Linux / OS X: <br>
./Scala-alapok.sh
